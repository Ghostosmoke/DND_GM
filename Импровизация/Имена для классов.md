> [!tip] 💡 Совет для ГМа
> Комбинируй класс с расой для мгновенного лора! Например:
> - **Гном-Изобретатель** = "Профессор Блим Шестеренка"
> - **Полуорк-Варвар** = "Бьорн Железный Кулак"
> - **Тифлинг-Колдун** = "Малакай Клятвопреступник"
> 
> Имя + класс + раса = готовый NPC за 3 секунды!
