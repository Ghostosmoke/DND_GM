Рабочее пространство для подготовки и ведения настольной ролевой кампании Dungeons & Dragons с элементами кроссовера.

## 📖 О кампании

Это архив материалов для кампании, в которой пересекаются вселенные из аниме, комиксов Marvel/DC, видеоигр и поп-культуры. Проект служит для хранения лора, характеристик персонажей, карт локаций и заметок для подготовки к игровым сессиям.

## 📂 Структура репозитория

```
DND_GM/
├── 📂 История/
│   ├── Враги.md              # Описание противников
│   ├── Карта маяка.md        # Локация "Шепчущий маяк"
│   └── Препятствия в маяке.md # Механики и ловушки
│
└── 📂 Персонажи/
    ├── Batman.md
    ├── Naruto.md
    ├── Kratos.md
    ├── Goku.md
    └── ... (60+ персонажей)
```

## 🛠️ Использование

Этот репозиторий предназначен для личного использования Гейм-мастером. Файлы в формате Markdown содержат:
- Заметки для подготовки к сессиям
- Характеристики NPC и боссов
- Описания локаций
- Сюжетные зацепки

## 📝 Добавление новых персонажей

Чтобы добавить нового персонажа в кампанию:
1. Создайте новый `.md` файл в папке `Персонажи/`
2. Используйте имя персонажа как название файла
3. Включите: краткое описание, характеристики, способности, связь с сюжетом

## 🎯 Планы

Список персонажей, которых планируется добавить в кампанию, находится в файле:
`Персонажи/Персонажи который я хочу добавить.md`
